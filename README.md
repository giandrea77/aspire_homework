# Subdocument array mutation

git clone https://github.com/giandrea77/aspire_homework.git

### Test using API

To set an environment variable in Windows:

```
SET NODE_ENV=development
```
on macOS / OS X or Linux:

```
export NODE_ENV=development
```

To start the sample:

```
npm run start
```
